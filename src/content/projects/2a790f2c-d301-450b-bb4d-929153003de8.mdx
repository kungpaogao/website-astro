# Here: this website


This website is an experiment using [Notion](https://www.notion.so/) as a [CMS](https://en.wikipedia.org/wiki/Content_management_system). This allows me to have a much better editing experience, which should encourage me to keep the content more up to date.

The process for updating content is easy as editing in Notion and triggering a rebuild/redeploy.

The site is built using [Astro](https://astro.build/), which statically generates the site and helps me minimize the amount of JavaScript that’s sent to the client.

You can find the [source code to this site on GitHub](https://github.com/kungpaogao/website-astro).

<br />


Below is what I used to test the Notion parser that I “wrote” (copied a lot of code from existing solutions) to convert Notion blocks to Markdown, which is then rendered using Astro’s built-in support for Markdown (underneath the hood, it uses <code>remark-rehype</code>).
